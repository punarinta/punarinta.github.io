web

This cocoon is intended to be used mostly with web-version of Bombyx, most of the functions will show an error when run in CLI. NB: function description is very brief, for now it's almost just a list.

  • render(filename, [variables]) — outputs template
  • secret(string, strength | hash) — generates/checks a password
  • cookie(name, [value], [options]) — reads/writes a cookie
  • session([id]) — creates/loads a session
  • fromGet(keyname) — gets a value of GET key
  • fromPost(keyname) — gets a value of POST key
  • fromJson([apath]) — gets a variable from posted JSON

text

A cocoon for working with text.

  • split(string, separator) — splits a string into an array using separator
  • trim(string) — trims a string
  • replace(string, old, new) — performs global replacement
  • length(string) — computes length of a string in symbols (not in bytes)

db

A cocoon for working with a database.

  • connect(host, user, pass, db, port) — connects to a database
  • row(link, query) — automatically gets a row from DB in a form of an array
  • query(link, query) — executes a query
  • fetchRow(sql_result) — manually extracts a row from sql-result

json

Utilitary JSON cocoon.

  • to(variable) — converts a variable into a string
  • from(string) — converts a string into a variable
